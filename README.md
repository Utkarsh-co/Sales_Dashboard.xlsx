# Sales Tracking Dashboard (Excel)

This is a beginner-level Excel dashboard created to analyze and track sales performance.

## Features
- Interactive filters using slicers (Year & Region)
- KPIs: Total Sales, Total Orders, Top Region
- Sales analysis by Region, Category, Segment, and Ship Mode
- Built using Pivot Tables and Pivot Charts

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
